# Transdiagnostic testing

Investigating the relationship between state engagement variability (SEV) and cognitive control in transdiagnostic populations 

(Related manuscript: "Flexible brain state engagement predicts cognitive control transdiagnostically")

**Within this folder**

**within_dataset.m** -- examine the relationship between SEV and cognitive control within each transdiagnostic dataset

**btw_dataset.m** -- predict cognitive control using SEV in previously unseen individuals 

**cpm_sev.m** -- identify brain networks supporting SEV using 10-fold connectome-based predictive modeling (CPM)

**cpm_cogcontrol.m** -- identify cognitive-control-predictive networks using 10-fold CPM 

**edgets_interactions.m** -- examine temporal alignment in SEV and cognitive control network dynamics

**CPM-master** -- folder containing functions necessary for CPM  

**Other information**

To extract SEV, we used recurring brain states identified in the Human Connectome Project dataset (neurodevelopment/centroid.mat) and steps detailed in sev.m
